Reads the status of the buttons of the two joypads. 

Takes two pointers for 16-\/bit bitmasks, one for the first player\textquotesingle{}s joypad, and one for the second one. Their bits are updated to show which buttons were pressed.

This function only supplies basic functionality, adequate for a normal digital pad.

If more advanced functionality is desired, use \hyperlink{psx_8h_a3375af2214d2bb2d8694e65c586d3fa4}{P\+S\+X\+\_\+\+Poll\+Pad()} instead of this function. \begin{DoxyAttention}{Attention}
Note that some joypads, like the official ones from Sony, do not like to be polled more than once every 1/60th of a second and if this limitation is not considered the data they return is undefined. Other joypads do not have this limitation but in any case err on the safe side. 
\end{DoxyAttention}

Sets an handler function for the V\+Blank interrupt. 

Used for simple, inaccurate timing -\/ the handler function gets called 60 times a second in N\+T\+SC video mode and 50 times a second in P\+AL video mode.

While most games use the V\+Blank interrupt for timing as they don\textquotesingle{}t require high precision and have mechanisms to keep up with the different speed in P\+AL or N\+T\+SC video mode, for precise timing V\+Blank is inadequate. It is better to look at root counters if you desire precision.

If there is already a V\+Blank handler set, this function replaces the current handler with the specified one.
