{\footnotesize\ttfamily \hyperlink{structModMusic}{Mod\+Music}$\ast$ M\+O\+D\+Load\+Ex (\begin{DoxyParamCaption}\item[{void $\ast$}]{d,  }\item[{unsigned int}]{flags }\end{DoxyParamCaption})}



Allocate a \hyperlink{structModMusic}{Mod\+Music} structure and copy data to it from data in memory containing a music module file. 

Almost all data from the music module file is copied into another location in memory for the \hyperlink{structModMusic}{Mod\+Music} structure.

This means your free memory should be roughly the double of the size of the M\+OD you\textquotesingle{}re loading.

You can avoid loading the samples in the module file\textquotesingle{}s native format, thus saving useful memory, by passing the M\+O\+D\+L\+O\+A\+D\+\_\+\+N\+O\+S\+A\+M\+P\+L\+ES flag. This is especially the case when you use \hyperlink{modplay_8h_a6ae6aa3cf9a0155265e289cdd1ae123c}{M\+O\+D4\+P\+S\+X\+\_\+\+Upload()}.
